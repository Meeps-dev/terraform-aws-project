# Meeps Terraform AWS Infrastructure

A hands-on AWS Infrastructure as Code project built with Terraform as part of my Cloud and Platform Engineering roadmap.

## Current Progress

- Installed and configured Terraform CLI
- Configured the HashiCorp AWS Provider
- Implemented input variables and default resource tags
- Created and reviewed Terraform execution plans
- Deployed and verified an Amazon S3 test resource
- Inspected local Terraform state
- Tested idempotency and infrastructure destruction

## Environment

- AWS Region: `eu-west-2`
- Environment: `dev`
- Infrastructure tool: Terraform
- AWS Provider: HashiCorp AWS Provider

> This repository will evolve into a modular AWS infrastructure project containing VPC, ALB, EC2, RDS, S3 and remote-state components.
